## Features

- **5 Difficulty Levels** with increasingly aggressive invaders
- **60 Physics Questions** covering:
  - Wave basics (frequency, wavelength, speed)
  - Intensity and amplitude relationships
  - Malus's Law and polarization
- **Retro Arcade Style** with particle effects and sound
- **Educational** - Learn physics while playing!

## Controls

- **Arrow Keys** or **A/D** - Move left/right
- **SPACE** - Shoot
- **Answer questions** when you hit enemies to score points

## Scoring

- Correct answer: **+100 points**
- Wrong answer: **-25 points**
- Player hit: **-50 points, lose 1 life**

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deploy to GitHub Pages

1. Fork this repository
2. Go to **Settings > Pages**
3. Set **Source** to "GitHub Actions"
4. Push to `main` branch - automatic deployment!
